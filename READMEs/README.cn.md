# ZX TCG 牌组转换工具

本工具用于从 [ZX TCG 官方网站](https://www.zxtcg.com/) 获取公开牌组，并将其转换为可用于 [Tabletop Simulator 模组](https://steamcommunity.com/sharedfiles/filedetails/?id=3642024821) 导入的卡牌列表。

或者可从`decks`资料夹複製官方预组的牌组列表使用（持续更新中）

## 使用方法
1. 前往[官方网站](https://www.zxtcg.com/)注册账号，并在「**マイデッキ一覧**」中创建新的牌组。
2. 将牌组设置为「**公開**」，并复制「**公開用URL**」。
3. 下载或复制`index.html`和`app.js`到同一文件夹中，并在浏览器中打开`index.html`。
4. 将「**公開用URL**」粘贴到输入框中，点击**Import Deck** ，然后复制生成的牌组列表。
5. 将牌组列表粘贴到 [Tabletop Simulator 模组]()的牌组导入器中即可完成导入。
